# ICM-20948 Anonymous V8 Diagnostic Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Port the STM32-compatible I2C/IMU interfaces to MSPM0G3507 and transmit a validated Anonymous Protocol V8 `ID=0x01` frame every 20 ms through `RESV1_UART`.

**Architecture:** `bsp_iic.c` provides the STM32-shaped `UserII2Dev` interface over MSPM0 DriverLib and owns all I2C transfer timing. `bsp_imu.c` provides the reference function-table API and a single ICM-20948 configuration/data path; the existing `ICM20948_*` API becomes a compatibility adapter. `ano_protocol.c` converts `IMU_DATA_t` into one 21-byte V8 frame, while `app/main.c` schedules sampling from the existing 10 ms timer.

**Tech Stack:** MSPM0G3507 DriverLib, SysConfig-generated `ti_msp_dl_config.*`, TI Arm Clang, C11 host tests with `C:\msys64\ucrt64\bin\gcc.exe`, Anonymous Communication Protocol V8.10.
